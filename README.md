# LUMA

App de bem-estar digital, foco e respiração em PT-BR.

## Rodar localmente

```bash
npm install
npm run dev
```

## Build de produção

```bash
npm run build
```

## Deploy na Vercel

1. Extraia este ZIP.
2. Envie os arquivos para o repositório no GitHub.
3. Na Vercel, importe o repositório.
4. Clique em **Deploy**.
